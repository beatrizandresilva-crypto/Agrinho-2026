1 - Descrição do projeto

Este projeto foi desenvolvido para o Concurso Agrinho 2026, na categoria Programação, subcategoria Ensino Médio. O site aborda o tema “Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”, apresentando reflexões e informações sobre a importância da tecnologia, da agricultura sustentável e da preservação ambiental no desenvolvimento do agronegócio brasileiro.
O projeto mostra como o uso consciente dos recursos naturais, aliado à inovação tecnológica no campo, pode aumentar a produção agrícola sem prejudicar o meio ambiente. Além disso, apresenta animações e elementos interativos desenvolvidos em JavaScript e p5.js para tornar a experiência mais dinâmica e educativa.

2 - Objetivo
O objetivo deste projeto é conscientizar os usuários sobre a importância do agronegócio sustentável, destacando práticas agrícolas responsáveis, preservação ambiental e o uso da tecnologia para melhorar a produção no campo sem comprometer os recursos naturais das futuras gerações.
Público-alvo
O projeto é destinado a estudantes, professores e comunidade em geral interessados em compreender a importância do equilíbrio entre a produção agropecuária e a preservação ambiental.
